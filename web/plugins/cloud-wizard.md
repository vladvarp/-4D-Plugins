## О плагине
[[ico:'image/CloudWizard/CloudWizard.png'-734]]
Cloud Wizard — процедурный генератор облаков для Cinema 4D. Создаёт кучевые и перистые облака с помощью метабол и шумовых материалов.

Плагин автоматически расставляет шарики метаболы внутри эллипсоида, подбирает профиль формы и опционально генерирует материал с шумами, рассеянием и подповерхностным свечением.

:::info
Плагин доступен через меню Extensions → Cloud Wizard. Не требует привязки к объектам — создаёт готовую группу в сцене.
:::

## Возможности

::::features
### ☁️ Пять типов облаков

`Кучевые (Cumulus)` Классические объёмные облака с вертикальной башнеобразностью.
`Перистые (Cirrus)` Тонкие вытянутые облака с прозрачностью.
`Грозовые (Cumulonimbus)` Высокие плотные облака с резким вертикальным ростом.
`Слоистые (Stratus)` Широкие плоские облака.
`Высококучевые (Altocumulus)` Средневысотные облака с умеренной пышностью.

### 🔧 Масштаб и форма
Общий масштаб (м), ширина (X), высота (Y), глубина (Z) и пышность (billowing) — полный контроль над формой облака.

### 🎲 Детализация метаболы
Количество шариков, порог метаболы (Hull), разброс и вариация размеров — от грубых форм до мелкой детализации.

### 🏷️ Профили типов
Каждый тип облака имеет готовый профиль параметров, который автоматически подставляется при выборе.

### 🎲 Рандомизация
Кнопки «Случайные формы» и «Случайный материал» для быстрого создания вариантов.

### 🌤️ Окружение
Опциональное добавление Physical Sky и направленного солнечного света.

## Использование

### Основной сценарий

1. Открой меню `Extensions` → `Cloud Wizard`
2. Выбери тип облака из выпадающего списка
3. Настрой масштаб, форму и детализацию
4. При необходимости включи «Создавать материал» и настрой цвета
5. Нажми **Создать облако**

:::tip
Для быстрого старта используй рандомизацию параметров через кнопки «Случайные формы» и «Случайный материал».
:::

### Параметры материала

Включите чекбокс «Создавать материал», чтобы плагин автоматически создал и применил материал к метаболе.

| Параметр | Описание |
|----------|----------|
| Основной цвет | Цвет освещённой части облака |
| Цвет тени | Цвет теневой части |
| Светимость (%) | Яркость канала Luminance |
| Прозрачность (%) | Прозрачность (по умолчанию для перистых — 35%) |
| Noise | Добавить шумовые слои на Color, Diffusion и Bump |
| Luminance | Дополнительная настройка свечения |

### Расположение

| Параметр | Описание |
|----------|----------|
| X | Горизонтальное смещение (м) |
| Y | Высота над сценой (м), по умолчанию 1500 |
| Z | Глубина смещения (м) |

### Дополнительно

| Параметр | Описание |
|----------|----------|
| Physical Sky | Добавить физическое небо (если его ещё нет в сцене) |
| Направленный свет | Добавить направленный источник «Sun Light» |

## Ограничения

- Облака строятся на метаболе — рендер зависит от настроек подразбиения (Subdivision)
- Кучевые и грозовые облака могут быть тяжёлыми при большом количестве шариков (55+)
- Плагин создает отдельный объект. Облака генерируются по одному.

:::warning
При большом количестве шариков (80+) время генерации и рендера может заметно возрасти. Сохраняй сцену перед тестом.
:::

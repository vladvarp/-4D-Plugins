## О плагине
[[mdb:'Интерфейс'-n'ID1'-b8]]
Floor Generator — процедурный генератор напольных покрытий для Cinema 4D, аналог Floor Generator из 3ds Max. Плагин берёт дочерний плоский полигон как границу и генерирует внутри него покрытие выбранного типа: шеврон, паркет, соты или ёлка.

Каждая плитка — это отдельный 3D-элемент с настраиваемой толщиной, фаской и швами. UV-координаты автоматически привязаны к размеру текстуры и поддерживают рандомизацию поворота, смещения и угла для каждого элемента. Плагин поддерживает отверстия для окон — отверстия вырезаются автоматически.

:::info
Плагин является генератором объектов — добавляется как дочерний элемент к плоскости, определяющей форму пола.
:::

## Возможности

::::features
### 🧱 Четыре паттерна

`Шеврон` Классический ёлочный паттерн с заполнением пустот (P1, P2).
`Паркет` Прямоугольная сетка со смещением рядов и разбросом ширины.
`Соты` Гексагональная сетка (flat-top).
`Ёлка` Паркетная кладка под 90° без пустот.

### 🔧 Толщина и фаска
Регулируемая толщина плиток и фаска (bevel) на кромках — от полного отсутствия до выраженного скоса.

### 📐 Швы
Включение/отключение швов и настройка их ширины. Швы создаются через `inset` полигонов каждой плитки.

### 🎲 Рандомизация UV
Настройка ширины и длины UV-тайлинга.
Поворот UV-координат.
Индивидуальный рандом угла UV для каждой плитки.
Рандом смещения UV по X и Y с отдельным сидом.

### 🔲 Дырки
Поддержка отверстий в поверхности — дочерний полигон может содержать внутренние контуры (окна, колонны). Плитки автоматически обрезаются по контурам.

### 🔀 Смещение паттерна
Глобальное смещение всего паттерна по X и Y для точного позиционирования.

### 🏷️ Теги выделения
Автоматические теги выделения: `P` — плитки, `S` — фаски/швы, `P1` и `P2` — заполнение (шеврон).
::::

## Использование

### Основной сценарий

1. Создай плоскость (`Plane`) нужного размера — это будет **граница пола**
2. Добавь объект `Floor Generator` в сцену
3. Сделай плоскость **дочерним элементом** Floor Generator
4. Настрой параметры в UserData плагина:
5. Выбери тип паттерна
6. Задай ширину и длину плитки
7. Отрегулируй толщину, фаску и швы
8. Результат обновится автоматически

:::tip
Для неправильных форм пола используй плоскость с произвольным контуром — плагин определит границу по крайним рёбрам меша.
:::

### Окна

Чтобы вырезать отверстие (окно, колонна, лестница), создай один полигон-плоскость с внутренним контуром. Плагин автоматически распознает внутренние петли как отверстия и обрежет плитки по их контурам.

:::tip
Для Г-образных или нестандартных отверстий используй нестандартный контур — плагин триангулирует невыпуклые дырки и корректно вычтет их из плиток.
:::

### Назначение материалов

Используй теги выделения для назначения разных материалов:

| Тег | Назначение |
|-----|------------|
| **P** | Основная поверхность плиток |
| **S** | Боковые грани и фаски |
| **P1** | Заполнение пустот (шеврон, тип 1) |
| **P2** | Заполнение пустот (шеврон, тип 2) |

## Ограничения

- Плагин работает только с **плоскими** дочерними объектами (минимум 3 вершины, количество палигонов не имеет значение)
- Для шеврона ширина плитки должна быть кратна высоте (встроена автоматическая коррекция)
- Количество генерируемых плиток зависит от размера плоскости — слишком большие площади могут замедлить работу

:::warning
При большом количестве плиток (тысячи) время генерации может составить несколько секунд. Сохраняй сцену перед первым тестом.
:::

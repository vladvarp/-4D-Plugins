## О плагине

ScatterPro — это мощный инструмент для создания реалистичных природных сцен в Cinema 4D.
Плагин позволяет распределять тысячи объектов по поверхности полигональной сетки с
точным контролем над каждым параметром: плотностью, вращением, масштабом и смещением.

Идеально подходит для создания травы, леса, каменных пейзажей, городских сцен
и любых других задач, где требуется быстрое заполнение пространства объектами.

## Возможности

::::features
### ✦ Умное распределение
Алгоритм Poisson Disk автоматически избегает пересечений между объектами при случайном распределении.

### ⚡ Высокая производительность
Обрабатывает миллионы полигонов и сотни тысяч инстансов без существенной потери FPS в вьюпорте.

### 🎨 Полная кастомизация
Контроль над плотностью, минимальным и максимальным масштабом, вращением по всем осям.

### 📦 Библиотека пресетов
24 готовых пресета: трава, камни, деревья, мох, цветы, городские объекты.

### 🗺️ Маски и веса
Используй Vertex Map, текстуру или нарисованную маску для управления областями распределения.

### 🔗 Интеграция с MoGraph
Полная совместимость с Cloner, Effectors и другими инструментами MoGraph.
::::

## Установка

1. Скачай архив `ScatterPro_v2.4.1.zip` по кнопке выше
2. Распакуй содержимое в папку плагинов Cinema 4D
3. Перезапусти Cinema 4D
4. Плагин появится в меню **Extensions → ScatterPro**

:::info
Папка плагинов Cinema 4D на Windows: `C:\Users\<имя>\AppData\Roaming\Maxon\Cinema 4D R2024\plugins`
На macOS: `~/Library/Application Support/Maxon/Cinema 4D R2024/plugins`
:::

## Быстрый старт

После установки создай базовую сцену со скаттерингом за три шага:

1. Выбери полигональный объект-поверхность в вьюпорте
2. Открой плагин через **Extensions → ScatterPro → New Scatter**
3. Перетащи объект для скаттеринга в поле **Source Objects**

:::tip
Для лучшей производительности используй Render Instances вместо обычных копий.
Включается в настройках через **ScatterPro Settings → Use Render Instances**.
:::

## Параметры

| Параметр          | Тип     | По умолчанию | Описание                              |
|-------------------|---------|-------------|---------------------------------------|
| `density`         | float   | 1.0          | Плотность распределения (0.0 – 10.0)  |
| `seed`            | integer | 42           | Зерно генератора случайных чисел      |
| `align_normal`    | boolean | true         | Выравнивать объекты по нормалям       |
| `min_scale`       | float   | 0.8          | Минимальный масштаб                   |
| `max_scale`       | float   | 1.2          | Максимальный масштаб                  |
| `rotation_random` | float   | 360.0        | Случайный поворот по оси Y в градусах |
| `use_mask`        | boolean | false        | Использовать текстурную маску         |

## Пример Python скрипта

Для автоматизации через Python SDK:

```python
import c4d
from c4d import plugins

def scatter_on_surface(surface_obj, source_objects, density=1.0):
    """
    Создаёт ScatterPro объект программно.
    surface_obj  — целевая поверхность
    source_objects — список объектов для распределения
    density      — плотность распределения
    """
    doc = c4d.documents.GetActiveDocument()

    # Создаём новый ScatterPro объект
    scatter = c4d.BaseObject(plugins.FindPlugin(1234567, c4d.PLUGINTYPE_OBJECT).GetID())
    scatter[c4d.SCATTER_DENSITY] = density
    scatter[c4d.SCATTER_SEED] = 42

    doc.InsertObject(scatter)
    c4d.EventAdd()
    return scatter
```

:::warning
Скрипт требует установленного плагина ScatterPro. ID плагина может отличаться
в зависимости от версии. Проверь актуальный ID в документации API.
:::

## Часто задаваемые вопросы

### Почему объекты проваливаются сквозь поверхность?

Убедись, что нормали целевого объекта направлены наружу. Выбери объект,
зайди в **Mesh → Normals → Align Normals**.

### Как использовать несколько исходных объектов?

Перетащи все объекты в поле **Source Objects** или помести их в нулевой объект
и укажи его как источник. Плагин будет случайно выбирать среди них.

### Поддерживается ли Cinema 4D Subscription?

Да, плагин совместим с Cinema 4D 2023 и 2024 (версии по подписке Maxon).

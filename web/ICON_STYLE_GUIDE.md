# C4D Plugin Icon Style Guide

## Формат
- 64x64 px, RGBA, прозрачный фон
- Рендер на 256x256 (SCALE=4), даунсемпл LANCZOS

## Цвета
```
CYAN   = (0, 200, 255)     — основной акцент, контуры
CYAN_L = (100, 220, 255)   — светлый, внутренние детали
CYAN_D = (0, 140, 200)     — тёмный, вторичные заливки
WHITE  = (230, 235, 240)   — узлы, блики
ORANGE = (255, 160, 50)    — стрелки, кнопки, маркеры
GREEN  = (80, 220, 130)    — галочки, пол
RED_L  = (255, 100, 100)   — удаление, крестики
YELLOW = (255, 210, 60)    — звёзды
GRAY   = (120, 130, 145)   — оси, нейтральные
GRAY_D = (70, 75, 85)      — тёмные заливки
```

## Принципы
- Фигуры занимают 80-90% canvas, без лишних полей
- Без рамок — чистый символ
- Геометрический flat-стиль, без градиентов
- Толщины: контур s(3), вторичный s(2), мелочи s(1)

## Координаты
`s(v) = v * 4`. Центр: `cx, cy = s(32), s(32)`

## Типовые символы
- Генераторы объектов — рисую саму геометрию
- Утилиты — шестерёнка, документ
- Теги — полигональная фигура тега
- Деформеры — деформированная геометрия
- Удаление — мусорное ведро + символ
- Оси — квадрат + стрелки + оси
- Анимация — таймлайн + стрелки
- Камеры — корпус + оптика

## Скрипт генерации
`web/generate_icons.py` — содержит функции `icon_<name>()` для каждой иконки. Новая иконка: пишу функцию, добавляю вызов в `main()`, запускаю `python web/generate_icons.py`. Результат в `web/ico/`.

Структура файла

Каждый .md файл плагина состоит из двух частей: frontmatter (мета-данные в начале файла) и тела документа (Markdown-контент).

Файл должен находиться в папке plugins/ и иметь расширение .md. Для открытия страницы плагина используется URL вида:

URL страницы плагина 
plugin.html?md=plugins/название-плагина.md

Frontmatter

Frontmatter — это блок с мета-данными в самом начале файла. Он обрамляется тремя дефисами --- сверху и снизу. Используется формат ключ: значение.

Формат frontmatter 
---
ключ: значение
другой_ключ: "значение со спецсимволами"
список: [первый, второй, третий]
---

Здесь начинается основной контент...

Обязательные поля

Эти поля обязательно должны быть заполнены — они используются в hero-секции страницы.

Поле Тип Описание Пример
title строка Название плагина ScatterPro
tagline строка Краткое описание (1–2 предложения) Мощный инструмент для скаттеринга
version строка Текущая версия v2.4.1
author строка Имя автора или команды Иван Петров
updated строка Дата последнего обновления 15 мая 2025
Пример обязательных полей 
---
title: ScatterPro
tagline: Мощный инструмент для распределения объектов по поверхности с полным контролем над плотностью и масштабом.
version: v2.4.1
author: Иван Петров
updated: 15 мая 2025
---

Необязательные поля

Дополнительные поля для заполнения боковой панели и hero-секции.

Поле Тип Описание Пример
icon символ Emoji или Unicode символ иконки ✦
tags список Категории плагина [Моделирование, Процедурный]
license строка Тип лицензии MIT
cinema4d строка Поддерживаемые версии C4D R21 — R2024
os строка Поддерживаемые операционные системы Windows, macOS
renderer строка Поддерживаемые рендереры Redshift, Arnold, Octane
download URL Ссылка на скачивание https://example.com/download
github URL Ссылка на репозиторий GitHub https://github.com/user/repo

Блоки Callout

Специальные блоки для выделения важной информации. Используется синтаксис :::[тип] … :::.

Доступные типы: info, warning, danger, tip.

Синтаксис callout блоков 
:::info
Информационное сообщение. Используй для подсказок и полезных ссылок.
:::

:::warning
Предупреждение. Используй для важных заметок, на которые нужно обратить внимание.
:::

:::danger
Критическое предупреждение. Используй для описания деструктивных действий.
:::

:::tip
Совет. Используй для лайфхаков и рекомендаций.
:::

Сетка фич

Для красивого отображения возможностей плагина используй блок ::::features. Каждая фича начинается с ###, затем иконка и название, следующая строка — описание.

Синтаксис сетки фич 
::::features
### ✦ Умное распределение
Автоматически избегает пересечений объектов при распределении.

### ⚡ Высокая скорость
Обрабатывает миллионы полигонов без потери производительности.

### 🎨 Кастомизация
Полный контроль над плотностью, масштабом и вращением.

### 📦 Пресеты
Готовые пресеты для трав, камней, деревьев и другой растительности.
::::

История изменений

Для отображения changelog используй блок ::::changelog. Каждая версия начинается с ### номер — дата, изменения — через дефис.

Синтаксис changelog 
::::changelog
### v2.4.1 — 15 мая 2025
- Исправлен краш при работе с объектами без полигонов
- Улучшена производительность на сценах с большим количеством объектов

### v2.4.0 — 3 апреля 2025
- Добавлена поддержка Cinema 4D R2024
- Новый алгоритм распределения по UV-координатам
- Исправлены мелкие баги в интерфейсе

### v2.3.0 — 10 января 2025
- Добавлены 12 новых пресетов растительности
- Поддержка Redshift Instancer
::::

## Установка

1. Скачай архив с плагином
2. Распакуй в папку `plugins` Cinema 4D
3. Перезапусти Cinema 4D

:::info
Папка plugins обычно находится по адресу: `C:\Users\<имя>\AppData\Roaming\Maxon\Cinema 4D\plugins`
:::

## 1. Модули Cinema 4D (c4d.*)

Основные модули для разработки плагинов, скриптов и автоматизации.

### c4d
Корневой модуль. Содержит основные константы, типы данных и функции API Cinema 4D.

**Использование:**
- Доступ к константам типов объектов (`Ocube`, `Ocamera`, `Ospline`, etc.)
- Типы данных (`DTYPE_LONG`, `DTYPE_BOOL`, `DTYPE_STRING`, etc.)
- Флаги и сообщения (`EventAdd()`, `MSG_UPDATE`, etc.)
- Работа с описаниями параметров (`GetCustomDataTypeDefault()`, `DescID`, etc.)

```python
import c4d
doc = c4d.documents.GetActiveDocument()
obj = doc.GetFirstObject()
print(obj.GetName())
c4d.EventAdd()
```

### c4d.gui

# VAr Tools

Набор утилит для [Cinema 4D](https://www.maxon.net/), созданных для ускорения повседневной работы. Все плагины находятся в меню **VAr Tools** в разделе **Расширения** (Extensions).

## Содержание

| Категория | Плагин | Описание |
|-----------|--------|----------|
| **OBJECT RENAMER PRO v2.4** | — | Плавающая немодальная панель с операциями переименования, выделения и сортировки объектов. Каждое нажатие кнопки создаёт один шаг отмены. |
| **Camera Resolution Manager v1.4** | — | Управление разрешением рендера для каждой камеры. |

### Axis

| Плагин | Описание |
|--------|----------|
| **Axis2Bottom v1.0** | Смещает ось (pivot) выделенных объектов в нижнюю центральную точку их bounding box (по X/Z — центр, по Y — нижняя грань). |
| **Axis2Center v1.0** | Смещает ось (pivot) выделенных объектов в геометрический центр их bounding box. |
| **AxisDrop v1.0** | Опускает ось (pivot) выделенных объектов на нижнюю грань их bounding box, сохраняя X и Z позицию нетронутыми. |

### Drop

| Плагин | Описание |
|--------|----------|
| **Drop2Floor v1.0** | Опускает выделенные объекты (и их иерархии) на уровень пола (Y = 0). |
| **Drop2Floor 0(XZ) v1.0** | Опускает выделенные объекты на уровень пола и центрирует их по осям X и Z. |

### Clean

| Плагин | Описание |
|--------|----------|
| **Clean Nulls v1.0** | Удаляет все Null-объекты без тегов из сцены. Дочерние объекты перемещаются на место Null-а, сохраняя мировые координаты. |
| **Clean Empty Nulls v1.0** | Удаляет пустые Null-объекты без тегов и без дочерних объектов. Работает в несколько проходов для вложенных цепочек. |

## Установка

1. Скопировать содержимое папки `VAr Tools` в директорию плагинов Cinema 4D:
   - **Windows**: `%APPDATA%\MAXON\Cinema 4D <версия>\plugins\`
   - **macOS**: `~/Library/Preferences/MAXON/Cinema 4D <версия>/plugins/`

2. Перезапустить Cinema 4D.

## Требования

- Cinema 4D R20+ (для плагинов Axis и Drop)
- Cinema 4D R26+ (для Camera Resolution Manager)

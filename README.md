# VAr Tools

Набор утилит для [Cinema 4D](https://www.maxon.net/), созданных для ускорения повседневной работы. Все плагины находятся в меню **VAr Tools** в разделе **Расширения** (Extensions).


## Содержание

| Плагин                        | Описание                                                                                                                                  |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------- |
| **Object Renamer PRO**        | Плавающая немодальная панель с операциями переименования, выделения и сортировки объектов. Каждое нажатие кнопки создаёт один шаг отмены. |
| **Camera Resolution Manager** | Управление разрешением рендера для каждой камеры.                                                                                         |


### Axis

| Плагин          | Описание                                                                                                                  |
| --------------- | ------------------------------------------------------------------------------------------------------------------------- |
| **Axis2Bottom** | Смещает ось (pivot) выделенных объектов в нижнюю центральную точку их bounding box (по X/Z — центр, по Y — нижняя грань). |
| **Axis2Center** | Смещает ось (pivot) выделенных объектов в геометрический центр их bounding box.                                           |
| **Axis Drop**   | Опускает ось (pivot) выделенных объектов на нижнюю грань их bounding box, сохраняя X и Z позицию нетронутыми.             |


### Location

| Плагин               | Описание                                                                        |
| -------------------- | ------------------------------------------------------------------------------- |
| **Drop2Floor**       | Опускает выделенные объекты (и их иерархии) на уровень пола (Y = 0).            |
| **Drop2Floor 0(XZ)** | Опускает выделенные объекты на уровень пола и центрирует их по осям X и Z.      |
| **Center2Parent XZ** | Центрировать по X и Z в позицию родителя (мировое пространство, Y не меняется)  |
| **Center2World XZ**  | Центрировать выделенные объекты по X и Z в мировом пространстве (Y не меняется) |


### Clean

| Плагин                    | Описание                                                                                                                 |
| ------------------------- | ------------------------------------------------------------------------------------------------------------------------ |
| **Clean Nulls**           | Удаляет все Null-объекты без тегов из сцены. Дочерние объекты перемещаются на место Null-а, сохраняя мировые координаты. |
| **Clean Empty Nulls**     | Удаляет пустые Null-объекты без тегов и без дочерних объектов. Работает в несколько проходов для вложенных цепочек.      |
| **Clean Objects**         | Удаляет все объекты того же типа, что и выделенный объект, сохраняя дочерние объекты.                                    |
| **Clean Empty Objects**   | Удаляет все объекты того же типа, что и выделенный объект, НО только если у них нет дочерних объектов.                   |
| **Clean All Tags**        | Удаляет все теги с выделенных объектов.                                                                                  |
| **Clean Select Tags**     | Удаляет все теги того же типа, что и выделенный тег. Поиск ведётся по всей сцене (по всем объектам).                     |
| **Clean Empty Mat-Tags**  | Удаляет все теги материалов у которых не назначен материал. Поиск ведётся по всей сцене (по всем объектам).              |
| **Clean Select Mat-Tags** | Удаляет все теги материалов с объектов, у которых назначен тот же материал, что и у выделенного тега материала.          |

### Tegs

| Плагин                    | Описание                                                                                                                                           |
| ------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Child Selector Teg**    | Тег выбора дочернего объекта.                                                                                                                      |
| **Target Camera**         | Создаёт камеру с целевой точкой (Null-объектом), на которую камера всегда смотрит. Таргет следует за камерой при переименовании.                   |
| **Camera Visibility Tag** | Тег управляет видимостью объекта (вьюпорт + рендер) в зависимости от активной камеры сцены (Stage-объект).                                         |


### Deformers

| Плагин             | Описание                                                                                                                  |
| ------------------ | ------------------------------------------------------------------------------------------------------------------------- |
| **PolySubdivider** | Эксперементальный аналог Divider: несколько алгоритмов разбиения полигонов с возможностью лёгкого добавления новых типов. |


## Установка

1. [https://github.com/vladvarp/-4D-Plugins/releases/tag/C4D](https://github.com/vladvarp/-4D-Plugins/releases/tag/C4D)

или

1. Скопировать содержимое папки `VAr Tools` в директорию плагинов Cinema 4D:
  - **Windows**: `%APPDATA%\MAXON\Cinema 4D <версия>\plugins\`
  - **macOS**: `~/Library/Preferences/MAXON/Cinema 4D <версия>/plugins/`
2. Перезапустить Cinema 4D.

## Требования

- Cinema 4D R20+ (для плагинов Axis и Drop)
- Cinema 4D R26+ (для Camera Resolution Manager)
